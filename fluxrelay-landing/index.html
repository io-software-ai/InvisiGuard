Capabilities核心能力

Everything a relay needs. In one key.一把金鑰，涵蓋中繼所需的一切。

Routing, resilience, tenancy, and billing: designed to work together, not stitched from four tools.路由、容錯、租戶隔離、計費，原生協同，而非拼裝四套工具。

Anthropic-native endpointAnthropic 原生端點

A drop-in ANTHROPIC_BASE_URL. Claude Code, the Anthropic SDK, and MCP tool-use work unchanged. No OpenAI translation layer in the way.直接設定 ANTHROPIC_BASE_URL；Claude Code、Anthropic SDK 與 MCP 工具呼叫原封不動可用，中間沒有 OpenAI 格式轉譯。

One key, every provider一把金鑰，串接全供應商

A single relay key fronts pooled upstream keys across providers, distributed by health and weight so no one limit ever gets tripped.單一中繼金鑰統一調度跨供應商的上游金鑰池，依健康狀態與權重分流，不會撞破任何單一限額。

Automatic failover自動容錯

On a 429 or any error, fluxRelay rotates keys and fails over across providers with backoff, before your users ever see a limit.遇到 429 或任何錯誤時自動輪替金鑰、跨供應商容錯並退避重試，在用戶撞到限流之前就處理好。

Multi-tenant by design原生多租戶

Issue scoped relay keys per team or customer, each with its own model allowlist, rate limits, and hard budget caps. One tenant can't starve another.為每個團隊或客戶簽發受限中繼金鑰，各自擁有模型白名單、速率限制與硬性預算上限；單一租戶不會排擠其他租戶。

Per-token billing按 token 計費

Meter every token, attribute usage per tenant, and pass costs through with margin. Billing is built in, not a spreadsheet you maintain.逐 token 計量、按租戶歸屬用量、加成轉嫁成本；計費是內建能力，不是你自己維護的試算表。

Yours to run部署在你這一側

Bring your own keys and run the relay on your own infrastructure. Your traffic and your margins stay with you.自帶金鑰、部署在你自己的基礎設施上；流量與利潤都留在你手中。

How it works運作方式

Point Claude Code at it and go.把 Claude Code 指過來就能跑。

Three steps from zero to a metered, multi-tenant relay.三步，從零到一個可計量、多租戶的中繼。

Create a relay key建立中繼金鑰

Add your upstream provider keys to a pool, then mint a scoped relay key with its own budget and model allowlist.把上游供應商金鑰加入池中，再簽發一把附帶預算與模型白名單的受限中繼金鑰。

Point your client指向你的客戶端

Set ANTHROPIC_BASE_URL and ANTHROPIC_AUTH_TOKEN. That is the entire integration: no SDK swap, no code change.設定 ANTHROPIC_BASE_URL 與 ANTHROPIC_AUTH_TOKEN，整合就完成了，不換 SDK、不改程式碼。

Ship and meter上線並計量

Every request is routed, failed-over, metered, and billed automatically. Watch usage per tenant in real time.每次請求自動路由、容錯、計量、計費，並即時看到每個租戶的用量。
